# ComfyUI-T2V-Project

A modular **Text-to-Video generation pipeline** built with **ComfyUI nodes** and **Wan 2.1 GGUF models**.

### Features
- Auto-downloads models from HuggingFace.
- Easy environment setup & dependency management.
- Fully interactive Gradio interface.
- Modular backend/frontend structure.

### Run Locally
```bash
python run_project.py
